# OpenGL_Graphics_3D_Scene
This is a project for school. It uses OpenGl in C++ to create a custom scene using simple shapes. I add objects, textures, lighting, and user input to traverse the scene and change the view.

When creating this project there was some foundational functionality to create simple shapes. For each shape it required the scale, rotation, position, textures, and shaders to be enetered. When I first started creating the project I noticed how this created a hard to read program with an abundance of redundant code. I refactored the code and made a function for each shape. This reduced many lines of code for one shape into 1 line. This made it easier to read and maintain.
